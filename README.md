\# SOC Brute Force Detection using Splunk



\## Overview

This project demonstrates detection of brute-force login attempts using Splunk SIEM.



\## Tools Used

\- Splunk Enterprise

\- Windows Event Logs



\## Detection Logic

\- EventCode 4625 (failed logins)

\- Threshold-based detection (count > 2)

\- Time-based correlation (5-minute window)



\## Dashboard Features

\- Authentication failure trends

\- Top targeted accounts

\- Brute-force detection alerts
